# アプリケーション概要

整体院向けの予約システム。

## 予約の流れ

1. メニュー選択
    - 自費
    - 保険適用
    - 初診
2. 日時選択
    - 1枠30分、4人
3. ログイン
    - Google
    - Facebook
    - Email/パスワード
4. 会員登録
    - 名前
    - カナ
    - Email
    - 電話番号
    - スポーツしてますか（チェックしたら表示）
        - 競技
        - 所属
5. 確認画面
6. 予約
7. メール通知
8. 初診時
    - 住所
    - タブレットで入力したい

## 管理者の設定項目

1. 店舗情報登録
2. スタッフ情報登録
3. 営業時間登録
4. メニュー登録
    - 名前
    - 概要
    - 画像
    - 時間枠は固定
    - 常に公開
    - 予約受付時間
    - キャンセル受付時間
    - 料金（自費のみ）
    - 自費／保険適用
